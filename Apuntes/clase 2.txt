# TEMA 1 - REGRESIONES LINEALES

## ¿Qué es una regresión?

Una regresión es un método estadístico que se utiliza para entender la relación entre diferentes variables. 
Su objetivo principal es determinar cómo cambia una variable dependiente (el resultado que queremos predecir) 
cuando una o más variables independientes (los factores que influyen) se modifican.

💡 EJEMPLO: Predecir consumo de café según la edad
- Variable independiente (X): edad
- Variable dependiente (Y): tazas de café
- La regresión encuentra la relación: Y = a + bX

---

## CONCEPTOS FUNDAMENTALES

**Conjunto de datos de entrenamiento (m):**
- m = número total de ejemplos de entrenamiento
- Ejemplo: 10,000 pares de (edad, consumo de café)

**Variables independientes (X's) - Características/Features:**
- x₁, x₂, x₃, ... los datos de entrada
- Ejemplo: edad de una persona

**Número de características (n):**
- n = cantidad de variables independientes
- En regresión lineal simple: n = 1 (solo una X)

**Variables dependientes (Y's) - Target/Salida:**
- Y = el valor que queremos predecir
- Ejemplo: tazas de café consumidas

---

## HIPÓTESIS (h)

La hipótesis es la función/fórmula que obtenemos del algoritmo de aprendizaje.
Compite con otros modelos para ver cuál predice mejor.

### Regresión Lineal Simple:

h(x) = a + bx

Donde:
- h(x) = valor estimado/predicho
- a = intersección (punto donde cruza el eje Y)
- b = pendiente (inclinación de la recta)
- x = variable independiente (entrada)

💡 TIP: El parámetro 'a' puede ser > 0, < 0 o = 0 (la recta NO siempre empieza de cero)
La recta puede empezar desde cualquier punto en el eje Y.

---

## FUNCIÓN DE COSTO (J) - Error Cuadrático Medio

La función de costo mide qué tan mal está prediciendo nuestro modelo. 
Cuanto MENOR sea, mejor será nuestro modelo.

### Fórmula del Error Cuadrático Medio (MSE):

J(a,b) = (1/2m) ∑ᵢ₌₁ᵐ (h(xᵢ) - yᵢ)²

Desglose:
- J(a,b) = error del modelo con parámetros a y b
- 1/2m = promedio (dividimos entre 2m para escala)
- ∑ = sumatoria para todos los m ejemplos
- (h(xᵢ) - yᵢ) = diferencia entre valor predicho y valor real
- (...²) = elevamos al cuadrado (para penalizar errores grandes)

### En términos simples:

J(a,b) = (1/2m) ∑ᵢ₌₁ᵐ (a + b·xᵢ - yᵢ)²

Objetivo: MINIMIZAR J(a,b) → encontrar los mejores valores de a y b

---

## ⚠️ ESCALADO DE CARACTERÍSTICAS (PASO OBLIGATORIO)

**IMPORTANTE: Todos los valores de X DEBEN estar en la MISMA escala ANTES de calcular θ**

Si NO escalas las características, el descenso por gradiente:
- Converge lentamente (tarda muchas más iteraciones)
- Puede diverdir/no encontrar la solución óptima
- Produce resultados inestables

### ¿Por qué? 

Si una característica va de 0-5000 y otra de 0-10:
- Los gradientes para θ₁ serán MUCHO más grandes que para θ₂
- El algoritmo "ignora" la variable pequeña

### Método recomendado: Estandarización Z-score

Para CADA característica X:

$$x'_i = \frac{x_i - \mu}{\sigma}$$

Donde:
- μ = media de X
- σ = desviación estándar de X

**Resultado**: Variables centradas en 0 con desviación estándar 1

### Ejemplo:

**Antes (sin escalar):**
- X₁ (edad): 15 a 75 años
- X₂ (ingresos): 20,000 a 150,000 euros

**Después (escalado):**
- X₁': -1.5 a 1.8 (aproximadamente)
- X₂': -1.2 a 1.9 (aproximadamente)

💡 Ahora están en la MISMA escala → descenso por gradiente funciona óptimamente

---

## DESCENSO POR GRADIENTE (Gradient Descent)

Es la estrategia principal para minimizar la función de costo J(a,b).
Se basa en ajustar iterativamente los parámetros para reducir el error.

### Algoritmo:

1. Inicializar: a = 0, b = 0 (o valores aleatorios)

2. Repetir hasta convergencia:
   
   a := a - α · ∂J/∂a
   
   b := b - α · ∂J/∂b

Donde:
- := significa "actualizar a"
- α (alfa) = learning rate / tasa de aprendizaje (velocidad de cambio)
   - Valores típicos: 0.01, 0.001, 0.1
   - Si α es muy grande → puede divergir (no converge)
   - Si α es muy pequeño → converge lentamente
- ∂J/∂a = derivada parcial de J con respecto a a
- ∂J/∂b = derivada parcial de J con respecto a b

### Derivadas parciales (Gradientes):

∂J/∂a = (1/m) ∑ᵢ₌₁ᵐ (h(xᵢ) - yᵢ)

∂J/∂b = (1/m) ∑ᵢ₌₁ᵐ (h(xᵢ) - yᵢ) · xᵢ

### Ecuaciones de actualización completas:

a := a - α · (1/m) ∑ᵢ₌₁ᵐ (a + b·xᵢ - yᵢ)

b := b - α · (1/m) ∑ᵢ₌₁ᵐ (a + b·xᵢ - yᵢ) · xᵢ

---

## RESUMEN DEL PROCESO (ORDEN CORRECTO)

**PASO 0 - ESCALADO (OBLIGATORIO)**: Normalizar todas las características X a la misma escala

1. **Dataset Sintético**: Generar datos aleatorios correlacionados (edad vs café)

2. **Escalado de características**: Aplicar Z-score a TODAS las variables
   - x' = (x - μ) / σ

3. **Inicialización**: θ₀ = 0, θ₁ = 0, θ₂ = 0, ...

4. **Iteración (Descenso por Gradiente)**:
   - Calcular predicciones: h(x) = θ₀ + θ₁x₁ + θ₂x₂ + ... (con X escalada)
   - Calcular error: J(θ) usando la fórmula MSE
   - Calcular gradientes: ∂J/∂θ₀, ∂J/∂θ₁, ∂J/∂θ₂, ...
   - Actualizar parámetros: θᵢ := θᵢ - α · ∂J/∂θᵢ

5. **Convergencia**: Repetir paso 4 hasta que J(θ) sea mínimo

6. **Resultado**: Obtener los valores óptimos de θ₀, θ₁, θ₂, ...

💡 ANALOGÍA: Imagina que estás en una montaña ciega. El gradiente te dice 
hacia dónde caminar para bajar más rápido. α es cuanto caminas en cada paso.
Sin escalado, algunos caminos son más "empinados" que otros aunque sea el mismo descenso.
